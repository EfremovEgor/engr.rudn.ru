Направления деятельности

  • Разработка рекомендаций по увеличению межсервисного интервала газотурбинных установок различной мощности на основании математического моделирования.
  • Оптимизация режимов эксплуатации энергетического оборудования тепловых электростанций.
  • Исследование критических деталей энергетического оборудования и разработка алгоритмов диагностики их технического состояния.
  • Разработка рекомендаций по снижению выбросов углекислого газа, оксидов азота на предприятиях электроэнергетики.
  • Разработка руководящих документов, стандартов организации, ГОСТов и других нормативно-технических документов для предприятий электроэнергетики.

Ресурсы

Лабораторная база центра позволяет проводить следующие работы и исследования:

  • решение общесистемных вопросов по структуре генерирующих мощностей в единой энергосистеме и изолированных энергосистемах;
  • решение проблем маневренности энергоблоков парогазовых установок и паросиловых установок, увеличения межсервисных интервалов газотурбинных установок на основании компьютерного моделирования;
  • согласование колебаний мощности возобновляемых источников энергии и особенностей их параллельной работы с тепловыми электростанциями;
  • разработка технологий для снижения выбросов дымовых газов тепловых электростанций.

В парке оборудования:

  • экспериментальная установка «Многоступенчатый тепловой насос» для оценки эффективности использования тепловых насосов при утилизации вторичных энергетических ресурсов;
  • стенд для измерения напряженно-деформированного состояния юбки составного поршня двигателя внутреннего сгорания;
  • гибкий USB видеоэндоскоп jProbe NT 1-85-80 SF для визуальной диагностики полых полостей различных агрегатов;
  • переносные газоанализаторы;
  • тепловизоры, пирометры;
  • портативные барометры, влагомеры, датчики температуры, анемометры;
  • течеискатели.

Команда центра решает широкий спектр задач в интересах академических и индустриальных партнеров по проведению научно-исследовательских работ по энергоэффективности оборудования тепловых станций и использованию вторичных и возобновляемых энергетических ресурсов, моделированию объектов и процессов, разработки новых технологий.

Результаты

  • Создана экспериментальная установка двухступенчатого теплового насоса (патент на изобретение) для оценки эффективности использования тепловых насосов при утилизации вторичных источников энергии.
  • Разработана и создана уникальная экспериментальная установка многоступенчатого теплового насоса (патент на изобретение), используемая для оценки эффективности использования тепловых насосов при утилизации вторичных источников энергии.
  • Разработан метод совершенствования системы охлаждения свободнопоршневого двигателя Стирлинга (СПДС) путем реорганизации течения рабочего тела в охладителе СПДС для улучшения эффективности теплоотвода и повышения технико-экономических показателей.
  • Разработана методика определения геометрических параметров исследуемых теплообменников, имеющих в качестве основного элемента конструкции сложенные медные ребра.
  • Разработана двухтрубная расчетная модель процессов теплообмена при течении рабочего тела в охладителе с использованием программного обеспечения ANSYS Fluent, на которой проведена валидация с использованием разработанного испытательного стенда
  • Разработана методика профилирования юбки составного поршня форсированного четырехтактного дизеля с учетом условий его смазки в цилиндре (патент на полезную модель, патенты на программы для ЭВМ) создан испытательный стенд.
